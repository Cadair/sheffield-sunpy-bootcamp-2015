SunPy Bootcamp

This website holds the material for the SunPy bootcamp given on the 26th-27th March 2015 at The University of Sheffield.

This bootcamp aims to teach you the basics of what you need to write good quality solar physics code in Python. It will introduce you to the Python programming language and then to the SunPy library.

Prerequisites

We assume that people working thorugh this material have some experience programming, and with working from a command line, but have minimal experience with Python.

Lessons

  1. Verson Control with git
  2. Stuart's IPython Notebook test
